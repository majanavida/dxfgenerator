# NotesHolder DXF Generator

Простой учебный Python-проект для генерации DXF-чертежа салфетницы NotesHolder. Пользователь задаёт основные размеры в окне tkinter, нажимает одну кнопку и получает раскладку деталей для лазерной резки.

## Ориентиры проекта

Перед разработкой были изучены ссылки из `useful.txt`:

- [задание по практике в Google Docs](https://docs.google.com/document/d/162a6-sGcsa88zUia3YB5BE5_bhKdqLJ7ODEPCwlt2EY/edit?usp=sharing);
- [каталог генераторов LaserBiz](https://box.laserbiz.ru/);
- [генератор детали NotesHolder / «Салфетница»](https://box.laserbiz.ru/NotesHolder).

Файл `NotesHolder.svg` из папки проекта использован как геометрический и визуальный образец. В нём видны отдельные плоские детали, прямоугольные шипы, внутренние пазы, пониженная передняя часть и компактная раскладка на листе. Учебная версия сохраняет эти основные признаки, но не копирует SVG пиксельно.

## Что генерируется

DXF содержит пять деталей:

1. дно с пазами под стенки;
2. пониженную переднюю стенку;
3. заднюю стенку;
4. левую трапециевидную боковину;
5. правую трапециевидную боковину.

На стенках предусмотрены простые прямоугольные шипы, на дне и боковинах — прямоугольные пазы с заданным зазором. Все контуры реза находятся на слое `CUT`. Текстовые подписи деталей в DXF не создаются.

## Технологии

- Python 3.10+;
- tkinter из стандартной библиотеки Python;
- ezdxf для записи DXF;
- pytest для базовых тестов.

## Установка и запуск

Откройте терминал в папке проекта и установите зависимости:

```bash
pip install -r requirements.txt
```

Запустите приложение:

```bash
python main.py
```

В окне уже стоят рабочие значения по умолчанию: 78×78×35 мм, материал 3 мм, открытость передней стенки 40%, шип 6 мм и зазор 0,1 мм. При необходимости измените их, укажите имя файла и нажмите **«Сгенерировать DXF»**.

Параметр **«Длина (внутренняя ширина)»** соответствует полю LaserBiz **«Секции гор.»**. Например, запись `78*1` на сайте означает одну горизонтальную секцию длиной 78 мм. В этой учебной программе поддерживается одна секция, поэтому вводится просто число `78`.

Файл сохраняется в `output/examples`. Расширение `.dxf` добавляется автоматически, если пользователь его не ввёл.

## Тесты

```bash
pytest
```

Тесты проверяют создание пяти деталей, размеры дна, понижение передней стенки, создание DXF и наличие слоёв `CUT` и `TEXT`.

## Упрощения

- Реализована только одна модель NotesHolder.
- Сложные скругления, ножки, декоративные вырезы и промышленные варианты соединений LaserBiz не повторяются.
- Криволинейное понижение стенок из SVG заменено прямой наклонной кромкой боковин и прямой низкой передней стенкой.
- Расчёт шипов и пазов намеренно простой: они равномерно распределяются по кромкам.
- Нет предпросмотра, Canvas, экспорта SVG/PNG и автоматической оптимизации раскладки.

## Ограничения

Программа выполняет только базовую проверку пустых полей и преобразование чисел. Она не является промышленным CAD-инструментом: перед реальной резкой размеры, зазор и раскладку следует проверить в DXF-редакторе и на тестовом образце материала.

## Структура

