# NotesHolder DXF/SVG Generator

Простое учебное приложение на Python с интерфейсом tkinter.

Программа строит раскладку пяти деталей салфетницы NotesHolder с прямоугольными шипами и пазами. Пользователь задаёт размеры изделия и может отдельными кнопками сохранить чертёж в форматах DXF или SVG.

Запуск:

```bash
pip install -r requirements.txt
python main.py
```

Готовые файлы сохраняются в папку `output/examples`.
